# PHP Programming Examples

![PHP](https://img.shields.io/badge/PHP-777BB4?style=for-the-badge&logo=php&logoColor=white)

A comprehensive collection of **PHP** programming examples covering fundamentals, algorithms, data structures, and more. Perfect for beginners and intermediate developers learning PHP.

## 📋 Overview

This repository contains **100+ individual PHP programs** organized by difficulty level and topic. Each file demonstrates a specific concept or problem-solving approach using PHP.

### Repository Structure
- All programs are standalone `.php` files
- Numbered sequentially (e.g., `001_01_01_Hello_World_Program.php`)
- Progressive learning from basic to advanced topics

## 🚀 Features

- Pure PHP console/web-ready scripts
- Covers core PHP concepts
- Clean and educational code structure
- Great for self-learning and interview preparation

## 📚 Program Categories

### Level 1: Basics
- Hello World
- Simple Calculator
- Even/Odd Check
- Area Calculations
- Basic File Handling

### Level 2-3: Control Structures & Functions
- Palindrome Checker
- Fibonacci Series
- Prime Number Generator
- String Manipulations
- Mathematical Operations

### Level 4-5: Intermediate
- Arrays & Loops
- OOP Basics (Classes & Objects)

### Level 6+: Advanced
- Data Structures (Stack, Queue, Binary Search Tree)
- Mathematical & Statistical Algorithms
- Form Validation Concepts
- Random Data Generation
- Advanced String & Number Operations

## 🛠️ How to Run

### Prerequisites
- PHP installed (`php --version`)
- Web server (optional, for browser testing)

### Run from Command Line
```bash
php filename.php
```

### Example:
```bash
php 001_01_01_Hello_World_Program.php
```

For web-based programs, place the file in your server's root directory (e.g., `htdocs`) and access via browser.

## 📖 Learning Path

1. Start with `001_01_01_Hello_World_Program.php`
2. Follow the numbering sequence
3. Modify the code to experiment
4. Combine multiple concepts to build small projects
